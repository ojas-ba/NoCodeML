# NoCodeML_backend

An application built using FASTAPI+Celery+Redis.
This application lets you build your ML project from scratch without any code, just Upload Dataset -> Choose Your own Preprocessing Steps -> select Model+Hyperparameters(Optional) and Click on Train.
